# GitHub Pages Setup

Trang web này được host tại: `https://honguyenquynhnhu84-prog.github.io/hotrothietkebaidaystem/`

## Cách deploy

1. Vào repository trên GitHub
2. Settings → Pages
3. Source: Deploy from a branch
4. Branch: `main`
5. Folder: `/docs`
6. Click Save

Trang web sẽ tự động được deploy trong vài phút!
